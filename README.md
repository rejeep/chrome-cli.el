# Chrome CLI for Emacs

This package provides an Emacs interface for
[chrome-cli](https://github.com/prasmussen/chrome-cli).

## Installation

Add `chrome-cli` to your [Cask](https://github.com/cask/cask) file:

```lisp
(depends-on "chrome-cli")
```
